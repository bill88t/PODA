## Building

Dependencies for the building prosses need

```js
npm install
```

Run debug preview with

```js
npm run dev
```

Create the distribute package in folder dist with

```js
npm run build
```

It can be reviewed with

```js
npm run preview
```

This run the dist folder as a static server
